# CODSOFT - Java Development Internship

Welcome to my **CODSOFT Java Development Internship** repository.

This repository contains the Java projects I completed as part of the CodSoft Java Development Internship Program.

## 📂 Tasks

### Task 1: Number Game

A console-based Java application where the user guesses a randomly generated number with hints and attempt tracking.

### Task 2: Student Grade Calculator

A Java application that calculates the total marks, average percentage, and grade based on the student's marks.

### Task 3: ATM Interface

A console-based ATM simulation that allows users to:

* Check account balance
* Deposit money
* Withdraw money
* Exit the application
